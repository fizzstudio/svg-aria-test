Svg elements: Graphics-document

This is a test using the graphics module specification, with role fallbacks, for an SVG structure. The root SVG uses role="graphics-document region", g elements use role="graphics-object group", and child elements use role="graphics-symbol img". No assistance has been given to a screen reader for navigation (such as via tabindex or JavaScript). We expect that the roles + aria should be sufficient.

    Expectations:
  • We expect that a screen reader (SR) can navigate to each element using a keyboard.
  • We expect that a SR will announce the aria-label correctly.
  • We expect that different SVG primitives (rect, circle, path, text) will be consistent with each other based on previous expectations.
  • We expect that the SVG elements will provide the same experience as the HTML elements (below the SVG space).

Interaction Instructions

This chart must be tested using a screen reader only. Navigate as normally when using a screen reader. The elements are not interactive and no javascript is used.
