SVG+ARIA Tests

A project testing ARIA in SVG. Includes atomic and integration tests, checked in different screen readers.

Testing ARIA roles, states, and properties from:

  • Accessible Rich Internet Applications (WAI-ARIA) 1.2
  • SVG Accessibility API Mappings
  • WAI-ARIA Graphics Module
  • Graphics Accessibility API Mappings

Current testing information

The overall plan for this repo is at this link. This includes the testing matrix for all ARIA as well as each test environment's ARIA as well as the screen reader + OS + browser matrix.

Testing progress is updated at this link.
